# gRPC Order Service

Сервис управления заказами на Go, реализующий CRUD-операции (создание, получение, обновление, удаление, просмотр) через gRPC.

## Особенности
- **gRPC API**: Строгая типизация и высокая производительность
- **Конфигурация**: Загрузка настроек из переменных окружения (`.env`)
- **Логирование**: Автоматическое логирование всех входящих запросов и времени их обработки через gRPC интерсепторы
- **Автоматизация**: `Makefile` для быстрой генерации кода, сборки и запуска

---

## Подготовка к использованию

### 1. Установка зависимостей
У вас уже должен быть установлен Go. Если его еще нет, обратитесь на https://go.dev/doc/install. 

Установите необходимые зависимости и плагины для работы с gRPC:
```bash
go mod tidy

# Установка плагинов для генерации кода (если еще не установлены)
go install google.golang.org/protobuf/cmd/protoc-gen-go@latest
go install google.golang.org/grpc/cmd/protoc-gen-go-grpc@latest
```
### 2. Настройка конфигурационного файла
Скопируйте пример файла окружения в рабочий файл `.env`:
```bash
cp .env.example .env
```
### 3. Использование команд `Makefile`
`Makefile` предлагает следующий функционал:

| Команда         | Описание                                                                                                      |
|-----------------|---------------------------------------------------------------------------------------------------------------|
| `make generate` | Генерирует файлы `*.pb.go` и `*_grpc.pb.go` из `.proto` определений. Используйте при изменении `.proto` файла |
| `make build`    | Компилирует проект в исполняемый бинарный файл (`order-service`)                                              |
| `make run`      | Выполняет сборку (`build`) и сразу запускает сервер                                                           |
| `make clean`    | Удаляет скомпилированный бинарный файл                                                                        |
| `make help`     | Показывает справку по доступным командам                                                                      |

Выберите необходимую команду и пользуйтесь!
